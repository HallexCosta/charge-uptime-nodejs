# Charge Uptime with Cluster Node.js

This example show how to node.js work with clusters in chaotic cases when app die because an error unexpected.

## Requirements
- Node.js (v17.5.0)
- Autocannon (v17.5.0)
- Yarn (v1.22.17)
- NPM (vl8.4.1)

## How to start?

````sh
## Entry cluster api
$ cd cluster-api/

## Start app
$ yarn start
`````

## Run charge tests

```sh
$ yarn test
```
